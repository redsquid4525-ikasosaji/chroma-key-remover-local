# Green Key Remover Local

画像をドラッグ＆ドロップすると、ブラウザ上だけで緑背景を抜いて透過PNGを作るローカルアプリです。

## 使い方

1. `index.html` をダブルクリックして開く
2. 緑背景の画像を画面へドラッグ＆ドロップ
3. 必要ならしきい値を調整
4. `PNGをダウンロード` を押す

## 構成

- `index.html`
- `style.css`
- `app.js`

サーバー不要です。外部ライブラリも使っていません。

## 処理内容

Pythonで使った緑抜き処理を、JavaScriptのCanvas処理へ移植しています。

- RGBから緑優位度を計算
- 明らかな緑背景は alpha = 0
- エッジの緑は段階的に透明化
- 緑フリンジを抑制
- アルファの小ノイズを3x3中央値で整理

## 注意

ブラウザの仕様上、処理結果は「ダウンロード」として保存します。元画像と同じフォルダへ自動上書き保存する挙動は、通常のHTML/JSだけでは行いません。
