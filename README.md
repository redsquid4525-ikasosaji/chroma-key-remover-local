# Green Key Remover Local

ブラウザだけで緑背景を抜き、透過PNGを作るローカルツールです。サーバーも外部ライブラリも不要です。

## 使い方

1. `index.html` を開く
2. 画像をドラッグ＆ドロップ
3. 必要なら緑抜き設定を調整
4. Transparent PNG上でクロップ範囲を調整
5. コピー、ドラッグ、またはPNGダウンロードで出力

## 主な機能

- 緑背景の透過処理
- 透明余白の自動クロップ
- 矩形ハンドルによる手動クロップ
- クロップ範囲のコピー、ドラッグ、PNGダウンロード
